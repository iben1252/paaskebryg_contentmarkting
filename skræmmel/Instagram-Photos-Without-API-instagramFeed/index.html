Options

Attribute Type Default Description
username Required String null Instagram username from where to retrieve the feed
container Required / Optional String null Selector where to place the feed. Required if get_raw_json is false
callback Required / Optional function null Function callback. Only applies if get_raw_json
get_raw_json Boolean false Enables getting raw json with a callback function
display_profile Boolean true Enables displaying the profile
display_biography Boolean true Enables displaying the biography
display_gallery Boolean true Enables displaying the gallery
styling Boolean true Enables default styling
items number [Int] 8 Num of items to display in gallery. Accepted values [0-12]
margin number [Float] 0.5 Margin between items in gallery
image_size number [Int] 640 Scale of items to build gallery. Accepted values [150, 240, 320, 480, 640]. Does not apply to videos previews.
host String [url] https://www.instagram.com/ URL where to fetch the data. Useful if instagram changes CORS policy

Example 1: Default feed styling

Don't like it? Check other examples!

Code:
<script src="jquery.instagramFeed.min.js"></script>
<script>
    (function($){
        $(window).on('load', function(){
            $.instagramFeed({
                'username': 'instagram',
                'container': "#instagram-feed1",
                'display_profile': true,
                'display_biography': true,
                'display_gallery': true,
                'get_raw_json': false,
                'callback': null,
                'styling': true,
                'items': 8,
                'items_per_row': 4,
                'margin': 1
            });
        });
    })(jQuery);
</script>
